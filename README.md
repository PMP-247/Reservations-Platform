# 🏢 Reservations Platform

A modern **resource reservation system** built with **React, TypeScript, and Node.js**.  
Book meeting rooms, projectors, or collaboration pods in real-time with live persistence and easy management.

---

## Features

- ✅ **Dynamic resource selection**: Meeting rooms, projectors, pods
- ⏰ **Time slot booking**: Book by half-hour slots
- 🖥️ **Live updates**: See confirmed reservations instantly
- 🧑‍💻 **User-friendly interface**: Built with React & TailwindCSS
- 🔒 **Safe environment**: Configurable with `.env` for API keys

---

## Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS, Lucide Icons  
- **Backend**: Node.js, Express, Supabase (or your preferred DB)  
- **Version Control**: Git & GitHub
